关于『地震不是由爆炸引起的』证明

客观事实

  • 东北地方太平洋近海地震是指2011年3月11日14时46分（当地时间）发生于日本东北地方外海三陆冲的矩震级9.0（USGS修正为矩震级9.1 [1]）的大型逆冲区地震。震中位于宫城县首府仙台市以东的太平洋海域，震源深度测得数据为24千米（14.9英里），并引发最大溯上高40.1米的海啸。此次地震是日本有观测纪录以来规模最大的地震，引起的海啸也是最为严重的，加上其引发的一系列灾害及核泄漏事故，导致大规模的地方机能瘫痪和经济活动停止，东北地方部分城市更遭受毁灭性破坏。


证明部分

下列部分相对独立，没有先后顺序

  • 1.日本天然气储量爆炸后的能量转换为爆炸当量
    日本天然气储量为：357亿立方米（35.7 bcm），数据来源[2]
    按照38/4200的比率折合为TNT爆炸当量（吨）计算 
    35.7 bcm *38/4200
= 323000000.0 t

    计算表达式：357*pow(10,8)*38/4200

    另：如果数据是现有产量总计，数据是29亿立方米（2.9 bcm）数据来源[2]
    按照38/4200的比率折合为TNT爆炸当量（吨）计算

    2.9 bcm *38/4200
= 26238095.23809524 t

    计算表达式：29*pow(10,8)*38/4200

  • 2.日本2011年（平成23年）3月11日东北地方太平洋近海地震直接转换为TNT当量
    将MW = 2/3 log10(MO) - 10.7的地震矩(MO)与矩震级(MW)的换算公式 [3]进行转换
    可的MO = [(MW + 10.7)*3/2]^10，单位为erg(尔格)，相当于10^-7 J
    代入计算 
    MO = [(9.1 + 10.7)*3/2]^10
 = 29.699999999999996^10
 = 5.011872336272674e+29 erg
 = 5.011872336272673e+22 J

    计算表达式：pow(10,(9.1 + 10.7)*3/2-7)
    现在我们将能量转换为TNT当量
    按照一吨TNT4.184×10^9 J的能量[4]计算

    5.011872336272673e+22 J /(4.184×10^9 J)
= 12082623761505.965 t

    计算表达式：5.011872336272673e+22/(4.148*pow(10,9))

  • 3.9.1级地震的能量转换为天然气
    将MW = 2/3 log10(MO) - 10.7的地震矩(MO)与矩震级(MW)的换算公式 [3]进行转换
    可的MO = [(MW + 10.7)*3/2]^10，单位为erg(尔格)，相当于10^-7 J
    代入计算 
    MO = [(9.1 + 10.7)*3/2]^10
 = 29.699999999999996^10
 = 5.011872336272674e+29 erg
 = 5.011872336272673e+22 J

    计算表达式：pow(10,(9.1 + 10.7)*3/2-7) 现在我们按照一立方米天然气38M（焦耳）的能量将能量转换为体积计算

    5.011872336272673e+22 J / 38M J
= 1318913772703335.0 m^3

    得到：74167971875.38023（立方米）
    计算表达式：5.011872336272673e+22/(38*pow(10,6))


结论

  • 将1和2进行对比：
    将323000000.0（吨）(或12082623761505.965 t)和12082623761505.965（吨）对比，得到——就算整个日本的天然气全部进行爆炸，也不可能支持这一场地震的能量，而日本又前后发生了大量7.0级以上的特大地震。
    所以，『地震是由爆炸引起』的理论通过证明可得出是错误。

  • 将3和大概数据对比：
    如果大概数据是357亿立方米（35.7 bcm），数据来源[2] - 3的结论是大概数据的36944.3633810458倍。
    如果大概数据是数据是29亿立方米（2.9 bcm）数据来源[2] - 3的结论是大概数据的454797.8526563224倍。
    通过反向计算证明，『地震是由爆炸引起』的理论也可得出是错误的。
